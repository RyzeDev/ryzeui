Top Navs

The RyzeUI Top Nav allows you to place a horizontal navigation anywhere you like on your page.

Navbars are responsive components used primary for top level site navigation and we recommend they are used only once per page.

Accessibility

The top nav allows for full keyboard accesibility. In order to prioritze the website navigation please remember to use role="navigation" on your <nav> element. Additionally, if your page uses off-canvas navigation, add the attribute aria-hidden="true" to one of themâ€”otherwise, the user's device will tab through (and possibly read) both of them before reaching any content.

Helper Classes

The top nav can easily be customised to suit your themes needs. To remove the site name from your navigation, just ommit the <div class="nav-header"> and the site name wil be removed.

Dropdown Menus

Dropdown menus help keep your nav bars clean and tidy. For functions such as logging out this UI element can be extremely useful.

Demo

HTML

<nav class="nav-bar nav-inline pull-right">
  <div class="nav-header">
    <a href="#" class="name"><span>RyzeUI</span></a>
  </div>
  <ul>
    <li><a href="#">Lorem</a></li>
    <li><a href="#">Lorem</a></li>
    <li><a href="#">Lorem</a></li>
    <li class="dropdown">
      <a href="#">Lorem</a>
      <ul class="dropdown-menu" role="menu">
        <li><a href="#">My Account and Settings</a></li>
        <li><a href="#">Settings</a></li>
        <li class="divider"></li>
        <li><a href="#">Logout</a></li>
      </ul>
    </li>
  </ul>
</nav>
