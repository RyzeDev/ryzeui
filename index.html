Getting Started

Getting started with RyzeUI is easy. Built on Sass for rapid deployment and is made for designers of all skill ranges.

What is RyzeUI?

RyzeUI is a "Responsive Front-End (Gaming) Framework" built for rapid theme deployment for developers using the Esportsify software to power their website.

This framework is designed to keep your front-end HTML short, semantic and sweet. It is also compatible with all modern web browsers and we are continually improving our framework thanks to its contributors.

RyzeUI is built by Matt Litherland and is maintained by the Esportsify Team.

Start Using RyzeUI

Getting started with RyzeUI is easy. We're currently available only on GitHub but will be including bower support shortly.

  • Clone the repo https://github.com/RyzeDev/ryzeui.git
  • Install with Bower bower install ryzedev/ryzeui

Autoprefixer required for Sass

RyzeUI uses Autoprefixer to deal with CSS vendor prefixes. If you're compiling RyzeUI from its Sass source and not using our full release, you'll need to integrate Autoprefixer into your build process yourself. We use CodeKit by bdkjones.

Normalize.css

RyzeUI uses Normalize.css to help make browsers render all elements more consistently and in line with modern standards. it precisely targets only the styles that need normalizing.
