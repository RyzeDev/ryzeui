Images

Images are commonly used throughout web content. They are great visual aids to support your content and therefore we have included a host of basic styles and alignment classes.

Basic Image

RyzeUI images by default are centered on the page. No class system is required to achieve this positioning.

Demo

HTML

<img src="image.jpg" alt="Image" width="300">

Responsive Image

RyzeUI provides a responsive image class. Note that this class will overwrite and image size attributes such as height and width. Use the class img-responsive to apply the responsive styling.

Demo

HTML

<img class="img-responsive" src="image.jpg" alt="Image">

Positioning Images
