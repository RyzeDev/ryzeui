Switches

Switches are commonly found in mobile applications and are now becoming more popular on the web. We have added our own basic switches to RyzeUI.

Accessibility

Forms utilise the browsers standard accessibility features.

Helper Classes

As shown in th demo above, RyzeUI allows you to control the size of your switches. By default the switches are considered to be a Medium size. You can add the sm class for small or the lg class for large to the label.

<label for="switchD" class="sm"></label>

<label for="switchD" class="lg"></label>
